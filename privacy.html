Privacy Policy

This application allows users to schedule and publish posts on Facebook Pages they manage.

Data We Collect

  • Facebook User ID
  • Facebook Page IDs
  • Page access tokens (stored securely and encrypted)
  • Scheduled post content created by the user

How We Use Data

Data is used solely to publish scheduled posts and comments on Facebook Pages authorized by the user.

Data Storage

Access tokens are encrypted and stored securely. We do not sell or share user data.

Data Retention

Data is retained only as long as the user account remains active or until deletion is requested.
